# FlightPlaner

FlightPlaner is a repository that contains code for a flight planning application. The application allows users to manage and search for flights based on various criteria.

## Features

- Flight Management: The application provides functionality to manage flights, including creating, updating, and deleting flight records.
- Flight Search: Users can search for flights using criteria such as departure airport, destination airport, and departure date.
- Object-Relational Mapping (ORM): It uses an ORM library (such as Entity Framework) to facilitate interaction with the database.


## Technologies Used

- C#: The primary programming language used in the repository.
- ASP.NET: The framework used for building the flight planning application.
